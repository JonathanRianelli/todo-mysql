## Instalação

### Client

  1. `$ cd client`
  2. `$ npm install` ou `$ yarn install`
  3. `$ npm start` ou `$ yarn start`
  
### Server

  1. Altere os dados do banco de dados mySQL em `server/index.js`
  2. `$ cd server`
  3. `$ npm install` ou `$ yarn`
  4. `$ npm start` ou `$ yarn start`
